# CodeOrbit AI VS Code Extension

CodeOrbit AI brings the same provider-agnostic coding workflow from the web app into VS Code.

## Commands

- `CodeOrbit AI: Open Panel`
- `CodeOrbit AI: Explain Selection`
- `CodeOrbit AI: Plan From Workspace`

## Supported providers

- Ollama
- LM Studio
- Anthropic
- OpenAI
- OpenRouter
- Any OpenAI-compatible endpoint
